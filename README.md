# 旋钮上位机调参工具 knob_tuner
基于 Python + Tkinter + pyserial 开发的电机旋钮串口可视化调参上位机，用于调试带自定义手感协议的硬件旋转编码器。

## 功能一览
1. 串口管理
- 一键刷新本机串口列表，自定义波特率（默认9600）
- 独立线程异步读取串口数据，UI不卡顿
- 连接/断开自动释放串口资源

2. 可视化参数面板
### CFG 手感配置
档位角度、吸附力度、限位力度、跳档阈值、PID阻尼参数全部可视化输入
### RANGE 数值区间
限制旋钮最大/最小档位，同步设置当前定位值
### DETENTS 特殊吸附点位
自定义多个吸附档位，支持一键清空吸附配置

3. 指令交互
- 各模块一键下发参数指令
- 自定义串口命令输入框，自由发送协议
- 读取设备全部参数 PRINTCFG、退出调参模式 EXIT_HID

4. 实时串口日志
区分发送(>)、接收(<)日志，支持一键清空日志面板
设备回传CFG参数时自动回填界面输入框，双向同步

## 运行环境
Python 3.7 及以上版本

## 依赖安装
### 方式1：通过 requirements.txt 批量安装
```bash
pip install -r requirements.txt
```

### 方式 2：手动安装依赖
```bash
pip install pyserial
```
## 完整使用流程
1、将旋钮硬件通过 USB 串口连接电脑
2、打开软件，点击【刷新串口】，选择对应设备端口
3、设置和硬件匹配的波特率（默认 9600），点击【连接】
4、修改面板内各项参数，点击对应应用按钮下发至设备
5、点击【读取参数】可拉取设备当前配置，自动填充界面
6、调试完成可点击【退出 HID】离开调参模式
